## 何时使用

数据保存成功反馈页面

## API

```html
<SaveSuccess
      detail='请核对候选人信息的准确性，如发现输入有误请及时修改。'
      columns=''
      data=''
/>
```


| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 主要信息 | string | '保存成功' |
| detail | 详情说明信息 | string | - |
| columns | 传入表格的表头 | [] | [] |
| data | 传入表格的数据 | [] | [] |
| className | 容器样式控制 | string | - |
